# コマンド集

## Git

```bash
# リモートリポジトリの一覧
git remote -v

# リモートリポジトリの追加
git remote add <リポジトリ名> <リポジトリURL>

# スタッシュに入れる
git stash push -m "READMEとか"

# スタッシュの一覧表示
git stash list

# スタッシュの適用(index0の場合)
git stash pop stash@{0}

# ブランチ作成と同時に切り替え
git switch -c <ブランチ名>

# リベース（指定したブランチを現在のブランチのベースとする）
git rebase <ベースにするブランチ名>

# gitのブランチログを表示
git log --graph --oneline --all --decorate
```
