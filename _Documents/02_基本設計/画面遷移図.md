# 画面遷移図（Mermaid）

作成日: 2026年1月31日
対象: Rust + Tauri によるLinux向けワイヤーフレーム作成アプリ

---

## 画面遷移フロー

```mermaid
flowchart TB
    Start([起動]) --> Home[ホーム/最近のファイル]

    Home -->|新規作成| NewFile[新規ファイル作成]
    Home -->|開く| OpenFile[ファイル選択]
    Home -->|設定| Settings[設定]

    NewFile --> Editor[エディタ/キャンバス]
    OpenFile --> Editor

    Editor --> Assets[アセット/ライブラリ]
    Editor --> Layers[レイヤーパネル]
    Editor --> Styles[スタイル/プロパティ]
    Editor --> Export[エクスポート]
    Editor --> Prototype[プロトタイプ]
    Editor --> Settings

    Export --> Editor
    Prototype --> Preview[プレビュー]
    Preview --> Editor
    Settings --> Editor

    Editor --> Save[保存]
    Save --> Editor

    Editor --> Close[閉じる]
    Close --> Home
```

---

## 画面イメージ（簡易ワイヤーフレーム）

Mermaidでの画面イメージは簡易表現になります。以下はエディタ画面のイメージです。

```mermaid
flowchart LR
    subgraph App[アプリケーションウィンドウ]
        direction TB
        TopBar[トップバー: ファイル/編集/表示/ヘルプ] 
        subgraph Body[メイン領域]
            direction LR
            Left[左サイドバー: ツール/アセット]
            Canvas[中央: キャンバス]
            Right[右サイドバー: レイヤー/プロパティ]
        end
        Bottom[ステータスバー: ズーム/座標]
    end
    TopBar --> Body --> Bottom
```

> 画面イメージを本格的に出力する場合は、UIモックツール（Figma等）で作成し、画像として貼り付ける方法が一般的です。
